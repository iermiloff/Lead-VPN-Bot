# Open-Source VPN Lead & Referral Telegram Bot 🚀

Полностью открытый, прозрачный и готовый к независимому техническому аудиту Telegram-бот. Разработан на базе кастомизированного форка `ayeresell` для автоматизации приема заявок на запуск брендированных VPN-сервисов и управления двухуровневой партнерской программой с расчетами в экосистеме TON.

---

## 📊 Прозрачная финансовая модель (100% оборота)
Вся экономика платформы фиксирована, прозрачна и не содержит скрытых комиссий. Общий оборот (100% денег от подписок пользователей) делится на два основных потока:

1. **65% от общего оборота** — Всегда гарантированно идет владельцу запущенного VPN-сервиса (Админу канала). Его доля неприкосновенна.
2. **35% от общего оборота** — Поступает на счета нашего сервиса в качестве чистой выручки для покрытия ИТ-расходов и выплаты реферальных.

Внутри наших **35% выручки** действует округленная Web3-пропорция **60 / 30 / 10**:
- **60%** (или 21% от общего оборота) — Доля создателя платформы за обеспечение стабильной ИТ-инфраструктуры (московские промежуточные сервера, зарубежные мосты, поддержка софта).
- **30%** (или 10.5% от общего оборота) — Уходит прямому рефералу (Партнер Уровня 1), который пригласил админа.
- **10%** (или 3.5% от общего оборота) — Уходит партнеру Уровня 2 (тому, кто пригласил Партнера Уровня 1).

*При такой схеме Главный админ просто вписывает в бот сумму, которая реально поступила на его счета (те самые 35%), а бот сам идеально распределяет вознаграждения рефералам, исключая кассовые разрывы.*

## 💎 Интеграция с экосистемой TON & Антиспам
Бот ориентирован на блокчейн-сообщество TON и этичный маркетинг:
- **Валидация кошельков:** Партнеры не получают доступ к рекламным материалам и реферальным ссылкам до тех пор, пока не привяжут рабочий TON-кошелек (Tonkeeper, MyTonWallet, Wallet в Telegram).
- **Crypto Bot Ready:** Структура хранения данных полностью подготовлена для бесшовных массовых выплат одной кнопкой через API `@CryptoBot`.
- **Строгая антиспам-политика (Anti-Spam Policy):** Платформа на программном уровне предупреждает участников о запрете спам-рассылок (в ЛС, чаты, комментарии). Использование спама ведет к моментальной перманентной блокировке партнера менеджерами через CRM-интерфейс и аннуляции начислений.

## ⚙️ Ключевой функционал

1. **Интерактивный Lead-генератор:** Короткий диалог для будущих владельцев VPN с разделением трафика («Для себя и друзей» или «Для подписчиков канала»). Без тарифов на старте — условия обсуждаются лично после оценки медиа-ресурса менеджером.
2. **Двухуровневые Диплинки:** Автоматическая генерация реферальных ссылок вида `t.me/your_bot?start=partner_id`. Бот «на лету» выстраивает цепочку рефералов при клике, исключая ручной ввод промокодов.
3. **Мульти-админка с CRM-статусами:** Уведомления о новых лидах приходят всей команде менеджеров, указанных в конфигурации. Поддерживаются пошаговые статусы обработки: `🆕 Новая` ➔ `⚙️ В работе` ➔ `📩 Оффер отправлен` ➔ `🟢 Установлен` или `❌ Отказано`. Бот логирует в БД ID и никнейм сотрудника, изменившего статус сделки.
4. **Автоматический расчет и рассылка выплат:** 
   - По кнопке «Расчет выплат» бот пошагово запрашивает у Главного админа сумму его личной поступившей маржи (35% от оборота конкретного канала).
   - Мгновенно формирует ведомость выплат для админа с макропоказателями прибыли.
   - **Автоматически рассылает уведомления** всем партнерам с указанием начисленной суммы и адреса их кошелька для проверки.

## 🛠 Быстрый запуск на сервере Linux (Ubuntu)

### 1. Подготовка системы и клонирование кода
Подключитесь к вашему VPS-серверу по SSH и выполните команды:
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3-pip python3-venv git -y

cd /home
git clone https://github.com/iermiloff/Lead-VPN-Bot
cd vpn-lead-bot
```

### 2. Настройка виртуального окружения
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Настройка конфигурации окружения
В репозитории подготовлен демонстрационный файл параметров. Скопируйте его под именем `.env` и заполните своими данными:

```bash
cp .env.example .env
nano .env
```

Внутри файла укажите ваши параметры:
```env
BOT_TOKEN=ваш_токен_из_botfather
ADMIN_IDS=123456789,987654321
SUPPORT_LINK=https://t.me
```
*Примечание: Перечислите Telegram ID менеджеров через запятую без пробелов. Первый ID в списке автоматически признается **Главным админом** — только ему будет доступна кнопка финансовых расчетов.*

### 4. Настройка автозапуска 24/7 (systemd)
Создайте системную службу, чтобы бот работал непрерывно и автоматически перезапускался при сбоях:

```bash
sudo nano /etc/systemd/system/vpn-bot.service
```

Вставьте следующий текст:
```ini
[Unit]
Description=VPN Lead and Referral Telegram Bot
After=network.target

[Service]
User=root
WorkingDirectory=/home/vpn-lead-bot
ExecStart=/home/vpn-lead-bot/venv/bin/python bot.py
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```

Запустите службу:
```bash
sudo systemctl daemon-reload
sudo systemctl enable vpn-bot.service
sudo systemctl start vpn-bot.service

# Проверка статуса (должен гореть зеленым active (running))
sudo systemctl status vpn-bot.service
```
